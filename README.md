# 棱镜 · 智慧校园高精度室内导航系统

## 作品简介
基于真实校园建筑CAD数据（1栋、2栋、3栋）开发的轻量化室内导航系统，支持2D/3D切换、楼层筛选、课表导入与解析、路径规划及课前提醒。

## 技术栈
- MapLibre GL JS (免费开源)
- HTML5/CSS3/JavaScript ES6
- GeoJSON 多边形渲染
- LocalStorage 模拟后端

## 数据说明
- 房间数据来自 `data/HD_1F2F.js`，包含1-3栋1-2楼所有房间多边形坐标。
- 课表解析支持“1栋1楼103”、“1栋201”等教室格式。

## 运行方式
直接双击 `index.html` 或部署到静态服务器。

## 核心功能
- 真实多边形渲染，按类型着色
- 2D/3D 视角切换
- 楼层独立控制
- 点击房间显示详情
- 课表粘贴导入，自动提醒
- 跨楼层路径规划（基于楼梯节点）
- 分类筛选与搜索
